# Qesto UI prototype

Интерактивный Flutter-каркас внутренней части Qesto. Один проект запускается в браузере и служит основой для будущих приложений Android, iOS и desktop.

## Что уже работает

- вкладки «Бюджет», «Выгода» и «Накопления» с сохранением состояния;
- месяцы бюджета, лимит, интерактивная диаграмма и список счетов;
- купоны, акции и отслеживаемые товары;
- цель накопления, серия, игровая сцена и экраны-заглушки;
- состояния загрузки, ошибки и отсутствия данных;
- тестовые данные отделены от интерфейса репозиторием.

## Запуск в VS Code

1. Откройте терминал в папке `apps/qesto_app`.
2. Выполните `flutter pub get`.
3. Для браузера выполните `flutter run -d edge`.
4. Для выбора другого доступного устройства выполните `flutter devices`, затем `flutter run -d <имя устройства>`.

Если команда `flutter` пока не находится, перезапустите VS Code после добавления Flutter в `PATH` или временно используйте полный путь к `flutter.bat`.

## Проверки

```text
flutter analyze
flutter test
flutter build web
```

Основная точка входа — `lib/main.dart`. Демо-данные находятся в `lib/mocks/fixtures`, а интерфейс по функциональным разделам — в `lib/features`.
